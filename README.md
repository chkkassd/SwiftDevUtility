This tool provides some useful functions which can be often used .It extends some Swift Standard Library and create some useful structs.
## Requirement
* iOS 11+
* Swift 4.2
## Installation
`Cocoapods:`
```bash
pod 'SwiftDevUtility'
```
`Carthage:`
```bash
github "chkkassd/SwiftDevUtility"
```
